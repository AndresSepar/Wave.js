# Wave.js

<b>Install With CDN</b> 
https://cdn.jsdelivr.net/gh/PiethonCoder/wave.js/wave.js

or download the wave.js fine from the root of this repository.

  <pre>
    var wave = new Wave();
  </pre>
</code>
